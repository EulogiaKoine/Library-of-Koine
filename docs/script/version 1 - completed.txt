< 최소 기능 >
저장된 주소의 텍스트를 읽어 반환.

< 필요 요소 >
1. 텍스트를 가리키는 주소
2. 실행 시 주소를 통해 텍스트를 읽어와 반환할 함수

< 구조 >
1. Script : (meta)스키마를 기반으로 DB에서 스크립트를 읽어 테이블로 보관 및 관리/읽기용 인터페이스 제공.
 - get: 스크립트 정보 전체 불러오기
 - read: 스크립트 텍스트 정보 불러오기
2. Page : 주소를 저장하는 단위. Script를 통해 저장된 주소의 텍스트를 읽어오는 메서드를 지원한다
 - script: 스크립트 객체
 - read: 텍스트 읽기
